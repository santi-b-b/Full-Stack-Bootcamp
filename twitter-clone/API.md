# API Documentation - Twitter Clone\n\nComplete API reference for Twitter Clone backend endpoints.\n\n---\n\n## Base URL\n\n`\nhttp://localhost:3000/api\n`\n\nAll endpoints are relative to this base.\n\n---\n\n## Table of Contents\n\n- [Authentication](#authentication)\n- [Tweets](#tweets)\n- [Users](#users)\n- [Error Handling](#error-handling)\n\n---\n\n## Authentication\n\nAuthentication uses session-based cookies. After login, include `credentials: 'include'` in all fetch requests.\n\n### Register User\n\n**POST** `/auth/register`\n\nCreate a new user account.\n\n**Request Body:**\n`json\n{\n  \"name\": \"John Doe\",\n  \"userName\": \"johndoe\",\n  \"email\": \"john@example.com\",\n  \"password\": \"SecurePassword123!\"\n}\n`\n\n**Response:** `201 Created`\n`json\n{\n  \"_id\": \"507f1f77bcf86cd799439011\",\n  \"name\": \"John Doe\",\n  \"userName\": \"johndoe\",\n  \"email\": \"john@example.com\",\n  \"image\": \"/assets/default-avatar.png\",\n  \"bio\": null,\n  \"location\": null,\n  \"following\": [],\n  \"createdAt\": \"2026-03-27T10:30:00.000Z\"\n}\n`\n\n**Error Responses:**\n`json\n// 400 Bad Request - Username taken\n{\n  \"error\": \"Username already exists\"\n}\n\n// 400 Bad Request - Invalid email\n{\n  \"error\": \"Invalid email format\"\n}\n`\n\n**JavaScript Example:**\n`javascript\nconst registerUser = async (userData) => {\n  const res = await fetch('/api/auth/register', {\n    method: 'POST',\n    headers: { 'Content-Type': 'application/json' },\n    body: JSON.stringify(userData)\n  });\n  return res.json();\n};\n`\n\n---\n\n### Login User\n\n**POST** `/auth/login`\n\nAuthenticate user and create session.\n\n**Request Body:**\n`json\n{\n  \"email\": \"john@example.com\",\n  \"password\": \"SecurePassword123!\"\n}\n`\n\n**Response:** `200 OK`\n`json\n{\n  \"_id\": \"507f1f77bcf86cd799439011\",\n  \"name\": \"John Doe\",\n  \"userName\": \"johndoe\",\n  \"email\": \"john@example.com\",\n  \"image\": \"/assets/default-avatar.png\",\n  \"bio\": null,\n  \"following\": []\n}\n`\n\n**Cookies Set:**\n- `session`: HTTP-only session cookie (7 days expiry)\n\n**Error Responses:**\n`json\n// 401 Unauthorized - Invalid credentials\n{\n  \"error\": \"Invalid email or password\"\n}\n\n// 400 Bad Request - Missing fields\n{\n  \"error\": \"Email and password are required\"\n}\n`\n\n**JavaScript Example:**\n`javascript\nconst loginUser = async (email, password) => {\n  const res = await fetch('/api/auth/login', {\n    method: 'POST',\n    headers: { 'Content-Type': 'application/json' },\n    credentials: 'include',\n    body: JSON.stringify({ email, password })\n  });\n  return res.json();\n};\n`\n\n---\n\n### Get Current User\n\n**GET** `/auth/me`\n\nRetrieve current authenticated user.\n\n**Authentication:** âś… Required (session cookie)\n\n**Response:** `200 OK`\n`json\n{\n  \"_id\": \"507f1f77bcf86cd799439011\",\n  \"name\": \"John Doe\",\n  \"userName\": \"johndoe\",\n  \"email\": \"john@example.com\",\n  \"image\": \"/assets/default-avatar.png\",\n  \"bio\": \"Software Developer\",\n  \"location\": \"San Francisco, CA\",\n  \"following\": [\"507f1f77bcf86cd799439012\"],\n  \"createdAt\": \"2026-03-20T15:30:00.000Z\"\n}\n`\n\n**Error Responses:**\n`json\n// 401 Unauthorized - Not authenticated\n{\n  \"error\": \"Not authenticated\"\n}\n`\n\n**JavaScript Example:**\n`javascript\nconst getCurrentUser = async () => {\n  const res = await fetch('/api/auth/me', {\n    credentials: 'include'\n  });\n  if (!res.ok) return null;\n  return res.json();\n};\n`\n\n---\n\n### Logout User\n\n**POST** `/auth/logout`\n\nLogout and invalidate session.\n\n**Authentication:** âś… Required\n\n**Response:** `200 OK`\n`json\n{\n  \"message\": \"Logged out successfully\"\n}\n`\n\n**JavaScript Example:**\n`javascript\nconst logoutUser = async () => {\n  const res = await fetch('/api/auth/logout', {\n    method: 'POST',\n    credentials: 'include'\n  });\n  return res.json();\n};\n`\n\n---\n\n## Tweets\n\n### Create Tweet\n\n**POST** `/tweets`\n\nCreate a new tweet.\n\n**Authentication:** âś… Required\n\n**Request Body:**\n`json\n{\n  \"text\": \"Just launched my new project!\",\n  \"images\": [\"https://cloudinary.com/image1.jpg\"]\n}\n`\n\n**Response:** `201 Created`\n`json\n{\n  \"_id\": \"507f1f77bcf86cd799439013\",\n  \"text\": \"Just launched my new project!\",\n  \"author\": \"507f1f77bcf86cd799439011\",\n  \"images\": [\"https://cloudinary.com/image1.jpg\"],\n  \"likes\": [],\n  \"replies\": [],\n  \"retweets\": [],\n  \"createdAt\": \"2026-03-27T10:30:00.000Z\",\n  \"updatedAt\": \"2026-03-27T10:30:00.000Z\"\n}\n`\n\n**Validations:**\n- Text is required and max 280 characters\n- Images array optional, each URL must be valid\n\n**Error Responses:**\n`json\n// 400 Bad Request - Empty tweet\n{\n  \"error\": \"Tweet text cannot be empty\"\n}\n\n// 401 Unauthorized\n{\n  \"error\": \"Not authenticated\"\n}\n`\n\n**JavaScript Example:**\n`javascript\nconst createTweet = async (tweetData) => {\n  const res = await fetch('/api/tweets', {\n    method: 'POST',\n    headers: { 'Content-Type': 'application/json' },\n    credentials: 'include',\n    body: JSON.stringify(tweetData)\n  });\n  return res.json();\n};\n`\n\n---\n\n### Get All Tweets\n\n**GET** `/tweets`\n\nFetch all tweets with pagination.\n\n**Query Parameters:**\n- `page` (optional): Page number (default: 1)\n- `limit` (optional): Tweets per page (default: 20)\n\n**Response:** `200 OK`\n`json\n[\n  {\n    \"_id\": \"507f1f77bcf86cd799439013\",\n    \"text\": \"Just launched my new project!\",\n    \"author\": \"507f1f77bcf86cd799439011\",\n    \"images\": [],\n    \"likes\": [],\n    \"replies\": [],\n    \"createdAt\": \"2026-03-27T10:30:00.000Z\"\n  },\n  // ... more tweets\n]\n`\n\n**JavaScript Example:**\n``javascript\nconst getTweets = async (page = 1) => {\n  const res = await fetch(`/api/tweets?page=${page}`);\n  return res.json();\n};\n``\n\n---\n\n### Get Tweet by ID\n\n**GET** `/tweets/:id`\n\nFetch a specific tweet.\n\n**Response:** `200 OK`\n`json\n{\n  \"_id\": \"507f1f77bcf86cd799439013\",\n  \"text\": \"Just launched my new project!\",\n  \"author\": \"507f1f77bcf86cd799439011\",\n  \"images\": [\"https://cloudinary.com/image1.jpg\"],\n  \"likes\": [\"507f1f77bcf86cd799439012\"],\n  \"replies\": [],\n  \"retweets\": [],\n  \"createdAt\": \"2026-03-27T10:30:00.000Z\"\n}\n`\n\n**Error Responses:**\n`json\n// 404 Not Found\n{\n  \"error\": \"Tweet not found\"\n}\n`\n\n---\n\n### Update Tweet\n\n**PUT** `/tweets/:id`\n\nUpdate a tweet (author only).\n\n**Authentication:** âś… Required\n\n**Request Body:**\n`json\n{\n  \"text\": \"Updated tweet content\"\n}\n`\n\n**Response:** `200 OK`\n`json\n{\n  \"_id\": \"507f1f77bcf86cd799439013\",\n  \"text\": \"Updated tweet content\",\n  \"author\": \"507f1f77bcf86cd799439011\",\n  \"updatedAt\": \"2026-03-27T11:00:00.000Z\"\n}\n`\n\n**Error Responses:**\n`json\n// 401 Unauthorized - Not tweet author\n{\n  \"error\": \"Only tweet author can update\"\n}\n\n// 404 Not Found\n{\n  \"error\": \"Tweet not found\"\n}\n`\n\n---\n\n### Delete Tweet\n\n**DELETE** `/tweets/:id`\n\nDelete a tweet (author only).\n\n**Authentication:** âś… Required\n\n**Response:** `200 OK`\n`json\n{\n  \"message\": \"Tweet deleted successfully\",\n  \"_id\": \"507f1f77bcf86cd799439013\"\n}\n`\n\n**Error Responses:**\n`json\n// 401 Unauthorized - Not tweet author\n{\n  \"error\": \"Only tweet author can delete\"\n}\n\n// 404 Not Found\n{\n  \"error\": \"Tweet not found\"\n}\n`\n\n---\n\n### Like Tweet\n\n**POST** `/tweets/:id/like`\n\nLike or unlike a tweet.\n\n**Authentication:** âś… Required\n\n**Request Body:**\n`json\n{\n  \"userId\": \"507f1f77bcf86cd799439011\"\n}\n`\n\n**Response:** `200 OK`\n`json\n{\n  \"message\": \"Tweet liked successfully\",\n  \"liked\": true,\n  \"likes\": 42\n}\n`\n\n**Note:** Sending like request from same user toggles like/unlike.\n\n**JavaScript Example:**\n``javascript\nconst likeTweet = async (tweetId, userId) => {\n  const res = await fetch(`/api/tweets/${tweetId}/like`, {\n    method: 'POST',\n    headers: { 'Content-Type': 'application/json' },\n    credentials: 'include',\n    body: JSON.stringify({ userId })\n  });\n  return res.json();\n};\n``\n\n---\n\n## Users\n\n### Get All Users\n\n**GET** `/users`\n\nFetch all users.\n\n**Query Parameters:**\n- `page` (optional): Page number (default: 1)\n- `limit` (optional): Users per page (default: 20)\n- `search` (optional): Search by name or username\n\n**Response:** `200 OK`\n`json\n[\n  {\n    \"_id\": \"507f1f77bcf86cd799439011\",\n    \"name\": \"John Doe\",\n    \"userName\": \"johndoe\",\n    \"email\": \"john@example.com\",\n    \"image\": \"/assets/default-avatar.png\",\n    \"bio\": \"Software Developer\",\n    \"location\": \"San Francisco\",\n    \"following\": [],\n    \"createdAt\": \"2026-03-20T15:30:00.000Z\"\n  },\n  // ... more users\n]\n`\n\n**JavaScript Example:**\n``javascript\nconst searchUsers = async (searchTerm) => {\n  const res = await fetch(`/api/users?search=${searchTerm}`);\n  return res.json();\n};\n``\n\n---\n\n### Get User Profile\n\n**GET** `/users/:id`\n\nFetch specific user profile.\n\n**Response:** `200 OK`\n`json\n{\n  \"_id\": \"507f1f77bcf86cd799439011\",\n  \"name\": Found\n{\n  \"error\": \"User not found\"\n}\n`\n\n---\n\n### Update User Profile\n\n**PUT** `/users/:id`\n\nUpdate user profile (only own profile).\n\n**Authentication:** âś… Required\n\n**Request Body:**\n`json\n{\n  \"name\": \"John Doe Updated\",\n  \"bio\": \"Updated bio\",\n  \"location\": \"New York, NY\",\n  \"image\": \"https://cloudinary.com/avatar.jpg\"\n}\n`\n\n**Response:** `200 Responses\n\n#### 400 Bad Request\n`json\n{\n  \"error\": \"Missing required fields\",\n  \"details\": \"Email and password are required\"\n}\n`\n\n#### 401 Unauthorized\n`json\n{\n  \"error\": \"Authentication required\",\n  \"message\": \"Please log in to perform this action\"\n}\n`\n\n#### 403 Forbidden\n`json\n{\n  \"error\": \"Access denied\",\n  \"message\": \"You don't have permission to perform this action\"\n}\n`\n\n#### 404 Not Found\n`json\n{\n  \"error\": \"Resource not found\",\n  \"resource\": \"tweet\",\n  \"id\": \"invalid-id\"\n}\n`\n\n#### 500 Internal Server Error\n`json\n{\n  \"error\": \"An unexpected error occurred\",\n  \"message\": \"Please try again later\"\n}\n`\n\n### Handling Errors in JavaScript\n\n``javascript\nconst fetchWithErrorHandling = async (url, options = {}) => {\n  try {\n    const res = await fetch(url, options);\n    \n    if (!res.ok) {\n      const error = await res.json();\n      throw new Error(error.error || `HTTP Error ${res.status}`);\n    }\n    \n    return await res.json();\n  } catch (err) {\n    console.error('API Error:', err.message);\n    throw err;\n  }\n};\n``\n\n---\n\n## Authentication Flow\n\n### Login Flow (Example)\n\n`javascript\n// 1. User submits login form\nconst login = async (email, password) => {\n  const user = await fetch('/api/auth/login', {\n    method: 'POST',\n    headers: { 'Content-Type': 'application/json' },\n    credentials: 'include', // Important: includes session cookie\n    body: JSON.stringify({ email, password })\n  }).then(r => r.json());\n  \n  // 2. Store user in state/context\n  setUser(user);\n  \n  // 3. Redirect to dashboard\n  router.push('/home');\n};\n\n// 3. All subsequent requests include session cookie automatically\nconst getTweets = async () => {\n  return fetch('/api/tweets', {\n    credentials: 'include' // Cookie sent automatically\n  }).then(r => r.json());\n};\n`\n\n---\n\n## Rate Limiting\n\nRate limiting is not currently implemented.
